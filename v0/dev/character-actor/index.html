Game controls

move(X)
Move one or more spaces, where X is a string of characters indicating the direction to move; "l" means left, "r" means right, "u" means up, "d" means down. E.g., move("ull") means move one square up then two squares left. Use the * operator to repeat moves, e.g., move("r" * 5) means move right five times.
print(X)
Print into the grid square you are currently occupying, where X is a string containing a single character. E.g., print("a") means print the character "a" into the current grid square.
change()
Disguise yourself as a random character to evade the agents.
escape()
Escape after being captured by an agent.
speed(N)
Change the game speed, where N is the number of moves per second.
help()
Receive your mission briefing.
